## Repository of Solutions and Explanations of the logics used while solving Competitive Programming Problems.

The files shall have the code and the problem statement, along with the explanation of the logic used to solve the problem. There will also be a best solution for each problem that will be explained in detail.

The problems shall be classified and linked accordingly in the categories as following:
1. Arrays
2. Strings
3. Linked Lists
4. Stacks
5. Queues
6. Trees
7. Graphs
8. Dynamic Programming
9. Searching and Sorting
10. Bit Manipulation
11. Mathematics
12. Greedy Algorithms
13. Backtracking
14. Divide and Conquer
15. Recursion
16. Hashing
17. Heaps
18. Sliding Window
19. Two Pointers
20. Miscellaneous
